# tokoApp
Aplikasi yang dapat membeli / mencari produk maupun warung yang dijual oleh toko kelontong. Aplikasi ini dibuat untuk memudahkan pengguna dalam menjual kebutuhan pokok terutama pada toko kelontong untuk memajukan UMKM. Disini saya menggunakan React Native untuk sisi client dan Express js untuk sisi server. Backend : bit.ly/3nHOIko
